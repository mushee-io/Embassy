# Mushee Embassy Premium Demo

A premium Next.js demo for **Mushee Embassy** — a white/pink, YC-style company site and product shell inspired by the large-company structure of MultiversX.

## Included

- 11+ page navigation surface
- Prelanding page
- Product, communities, creators, teams, security, pricing, ecosystem, cases, grants, and about pages
- Main app dashboard
- 24 interactive demo modules
- White/pink premium visual system
- Mainnet access CTA surface
- Europe Global Incorporated footer and company framing

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion
- lucide-react

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

1. Unzip the project
2. Push to GitHub
3. Import the repo into Vercel
4. Deploy with the default Next.js settings

## Notes

This build is inspired by the **structure and feel** of MultiversX — large hero, network-style metrics, partner strip, use-case sections, news block, and enterprise footer — while using Mushee Embassy branding and a custom white/pink visual language.
