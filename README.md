# Mushee Embassy — MultiversX-inspired premium demo

A dark, enterprise-style Next.js demo for Mushee Embassy, rebuilt around the current MultiversX homepage rhythm:
- giant dark hero
- hard network metrics
- integrations / partner rail
- builders section
- ecosystem counts
- token / economic section
- community proof section
- latest news section
- full footer CTA
- deep main app with 24 interactive modules

## Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Framer Motion
- Lucide React

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub and import into Vercel.
