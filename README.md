# Mushee Embassy Demo

A deployable Next.js demo for **Mushee Embassy** with the exact flow:

- **Pre-landing page**
- **Landing page** with 24 interactive demo cards
- **Main app dashboard** with searchable/toggleable demo modules

## Stack

- Next.js App Router
- React 19
- Tailwind CSS
- Framer Motion
- lucide-react

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Notes

This version is aligned to the Mushee Embassy brief only.
It does not use external brand naming in the UI.
