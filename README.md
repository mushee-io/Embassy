# Mushee Embassy Premium Demo

Deployable Next.js + Tailwind CSS demo for **Mushee Embassy**.

## Included
- white and pink premium design system
- YC-style prelanding page and company surface
- 11+ landing/product sections within the app shell
- 24 interactive demo modules
- main app dashboard with filters, toggles, and preview panels
- ready-to-ship structure for Vercel deployment

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
npm run start
```

## Deploy on Vercel
1. Push this folder to GitHub
2. Import the repo in Vercel
3. Framework: Next.js
4. Deploy

No environment variables are required for this front-end demo.
